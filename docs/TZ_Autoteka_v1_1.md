# ТЗ v1.1 — «Автотека» (клиентский макет, данные замоканы)

Короткая версия: клиентское SPA на Vue 3.5 + Tailwind, список плиток магазинов, меню (город/категории/фишка), сортировка по правилам a/b/c/d, страница магазина с галереей и “доскролл вниз → переход на сайт”, 6 тем (3 стиля × 2 палитры), mobile-first.

Полная версия — в DOCX: `docs/TZ_Autoteka_v1_1.docx`.
