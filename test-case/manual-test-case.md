## Ручные

1. Каталог /:
    - смена города, категорий, фишки работает,
    - после перезагрузки (F5) город/категории/фишка восстановлены.
2. Карточка /shop/:id:
    - галерея работает (или пустое состояние корректно),
    - переход на сайт через кнопку и через overscroll работает.
    - после перезагрузки тема сохранена.
    - открытие/закрытие, Esc, возврат фокуса — без регрессий.
