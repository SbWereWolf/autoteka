# Автотека — клиентский макет (Vue 3.5 (Composition API) + Tailwind 4.1+)

Это скелет проекта под согласованное ТЗ: каталог плиток, меню (город/категории/фишка),
сортировка по алгоритму a/b/c/d, 6 тем (3 стиля × 2 палитры), страница магазина, переход на сайт
через кнопку и через "доскролл вниз".

## Запуск
1) Установить зависимости
   npm i
2) Старт
   npm run dev

## Где править данные
- src/mocks/dicts.json — справочники (города, категории, фишки, темы)
- src/mocks/shops.json — магазины (29 шт. = 17/3/9)

Файлы сохранены в pretty format для удобного ручного редактирования.

## Где логика сортировки
- src/utils/sortShops.ts


## Темы и цвета
- Цветовая система тем использует только `oklch(...)`.
