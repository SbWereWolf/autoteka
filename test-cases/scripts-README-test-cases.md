## Тест-кейсы

### TC-SCRIPTS-README-001

- Утверждение: `scripts/` содержит только общие утилиты репозитория,
  фронтовые `.mjs` находятся в `frontend/scripts/`.
- Проверка:
  1. Проверить состав файлов в `scripts/`.
  2. Проверить наличие `.mjs`-утилит в `frontend/scripts/`.
  3. Убедиться, что в `scripts/` нет фронтовых `.mjs`.
- Ожидаемый результат:
  разделение утилит по каталогам соответствует документу.
- Тип: automated
- Каталог тестов: `system-tests`
