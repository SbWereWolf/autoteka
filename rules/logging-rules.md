# Правила ведения журнала работы

- Запрос пользователя или план запиши в файл с отметкой времени, файл
  создай в [logs](logs), для имени файла используй шаблон
  {platform}-{model}-{UNIX timestamp}-{uuid}.md, где:
  - {platform} — платформа на которой ты работаешь (cursor, codex, etc.)
  - {model} — имя конкретной модели LLM
  - {UNIX timestamp} — текущий UNIX timestamp
  - {uuid} — значение сгенерированное в формате UUID
- Если выполнил запрос от пользователя и делаешь доклад, то запиши
  доклад в тот же файл, если файла не существует, то сначала создай
  его
